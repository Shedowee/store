# Infinity Styles

**Infinity Styles** is a modern, interactive online store built with **React** and powered by **JSON Server**.  
It includes a product catalog, categories, search, modals, toast notifications, authentication pages, and a full **Admin Panel** where you can add, edit, and delete products.
